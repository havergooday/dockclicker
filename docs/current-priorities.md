# 현재 우선순위

## 현재 단계

직접 파견 + 자동 파견 기본 루프 모두 완성.  
파츠 개별 아이템 시스템 리팩터링(v3) + 공작실 인벤토리 탭 구현 완료.  
PC 터미널 메뉴형 레이아웃 구현 완료.  
격납고 관리 허브 재설계 완료 (파일럿 배정, 머신 분해).  
파견 관제 패널 전면 재설계 완료 (anchor Tween 2-state 레이아웃, 전체 슬롯 스크롤). 다음 작업은 사운드 시스템.
