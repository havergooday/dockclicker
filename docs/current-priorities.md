### 2순위 — 자동 파견 기반

직접 파견으로 번 크레딧을 자동 파견 시스템에 투자하는 루프.

1. PC 터미널 — 파츠 구매, 파일럿 고용 UI
2. 공작실 패널 — 파츠 조합으로 머신 완성
3. 파견 관제 패널 — 임무 보드, 파일럿·행성 배정, 자동 파견 시작

### 3순위 — 확장

1. 저장/불러오기
2. 자동 재파견 및 전원 출격
3. 행성별 고유 생물 그래픽
4. 플레이어 기체 추가 업그레이드 (범위, 무기)
