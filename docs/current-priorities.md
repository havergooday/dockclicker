# 현재 우선순위

## 현재 단계

프로젝트는 `기획/디자인/구조 정리 완료` 이후, `첫 MVP 구현 착수 직전` 단계에 있다.

## 가장 먼저 할 일

1. 메인 씬을 실제 UI 구조로 구체화
- 가로형 기준 `Hangar / Operations Panel / Bridge Lounge` 를 더 구체적인 패널 구조로 바꾼다.

2. 직접 파견 MVP 루프 구현
- 파견 시작
- 파견 중 클릭
- 복귀
- 정산

3. 수익 상태 저장 위치 결정
- 전역 상태 오토로드 기반으로 갈지
- 저장 파일 구조를 같이 열지 결정

## 높은 우선순위

- 중앙 운영 패널 최소 기능 구현
- 직접 파견 상태 표시
- 크레딧 증가/정산 반영
- 브리지 구역에 최소 상태 텍스트 배치

## 중간 우선순위

- 자동 파견 슬롯 뼈대
- 머신 슬롯 UI
- 파일럿 상태 카드
- 세로형 대응 구조 초안

## 낮은 우선순위

- SD 캐릭터 애니메이션
- 꾸미기 요소
- 스팀 배포 준비 세부 설정
- 고급 최적화

## 작업 시 주의

- 지금은 구조를 크게 흔들기보다 `작은 MVP 하나를 끝까지 연결`하는 편이 좋다.
- 문서만 늘리고 실제 플레이 루프를 늦추지 않도록 주의한다.
- 실제 구현이 들어가면 문서 상태도 같이 갱신한다.
