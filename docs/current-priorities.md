# 현재 우선순위

## 현재 단계

직접 파견 + 자동 파견 기본 루프 모두 완성.  
파츠 개별 아이템 시스템 리팩터링(v3) + 격납고 조립 팝업/인벤토리 구현 완료.
PC 터미널 메뉴형 레이아웃 구현 완료.  
ShipCanvas (가로 파노라마 함선 캔버스) 구현 완료.  
항성지도 팝업 전면 재설계 완료 — 행성 슬롯 4행 그리드, 베이 팝업 슬라이드인, 우클릭 드래그 스크롤.  
직접 파견 클리커 UX 개선 완료 — "미션완료" / "함선복귀" 플로팅 텍스트.  
격납고 구역 (`HangarZone`) 구현 완료 — 4격납고×4베이 구조, 오버레이 팝업 방식 상세, 격납고 잠금/해금.  
세이브 v4 전환 완료 — hangar_groups 직렬화, v3→v4 마이그레이션 내장.

현재 작업은 **UI/UX 통합 재설계** Phase 2~3 진행 중.  
격납고 구역화 + 베이 팝업 전역 오버레이 전환 작업 완료. 다음은 격납고 팝업 세부 폴리싱과 관제실 팝업 정리.

### 2026-05-29~30 작업 (게임플레이 컨텐츠 확장)
- 레거시 플립 패널 4종 제거 + `ship_canvas` / `hangar_zone` 코드 분리 리팩터링
- 클리커 강화 시스템: 파츠 상점에 **강화 탭** 추가 (클릭 데미지 / 자동공격 / 클릭 범위 / 연타 콤보)
- 클리커 생물 6종 + 이동 패턴 (좌우/상하/원/반사/8자/불규칙), 행성 난이도별 티어 매핑
- 파츠 드랍 시스템 (40% 확률) + 드랍 파츠 옵션 2개 부착 / 효과 적용 / 분해 환급
- 베이 슬롯 이름 부여 (`custom_name`)
- 하부 데크(숙소) 골격 + 우클릭 수직 스냅 드래그 — **컨텐츠는 미정 (보류)**
- 옵션 팝업 (사운드) — nav bar ⚙ 버튼, 전체 슬라이드다운으로 팝업 연출 통일

사운드 SFX/BGM 트랙은 아직 미구현 (옵션 패널의 버스 슬라이더만 준비됨).

### 다음 후보 (`docs/ideas-30.txt` 참고)
- 단계별 기능 해금 프로그레션 / 전원 출격 버튼 / 오프라인 다중 사이클
- 파일럿 컨텐츠 (상세·대화·출격 로그) — 숙소 데크 활용 방향 재논의 필요
- 크레딧 HUD 롤업 연출 / BGM·SFX

상세 계획은 `docs/plans/2026-05-27-ux-redesign.md`를 기준으로 한다.
