# 현재 우선순위

## 현재 단계

현재 단계는 기능 추가가 아니라 유지보수다.  
코드 기준선은 ShipCanvas / 항성지도 / 격납고 / 관제실까지 반영된 상태로 고정한다.

## 현재 할 일

1. 버그 수정
2. 레거시 씬/스크립트 삭제
3. 문서와 테스트 정합성 유지
4. 저장/이벤트/표시 텍스트 회귀 점검

## 고정 기준

- 직접 파견 + 자동 파견 기본 루프는 완료 상태다.
- 파츠 개별 인스턴스, 파일럿 시스템, 시설/기지 해금, Living Ship 상태는 현재 기준선이다.
- 앞으로는 새로운 기능을 추가하지 않고, 기존 기능을 정리하거나 제거한다.
