# 현재 우선순위

## 현재 단계

직접 파견 + 자동 파견 기본 루프 모두 완성.  
파츠 구매 → 조립 → 파견 → 귀환 → 수령 → 재투자 전체 사이클 동작.  
전 패널 UI 리뉴얼 완료 (상점·공작실·파견관제 모두 1600×420 가로형 최적화).  
다음 목표: 게임 루프 내구성 강화 (저장/불러오기, 자동 재파견).
