# 현재 우선순위

## 현재 단계

구현 진행 중. 패널 플립 내비게이션 + 도커형 창 구성 완료.  
다음 목표: 직접 파견 클리커 화면 (외계생물 → 클릭 데미지 → 크레딧 드랍).

## 논의 미완료 항목

없음. 격납고/정산 설계까지 전부 확정됨.

## 구현 우선순위

### 1순위 — MVP 플레이 루프

패널 플립 시스템과 직접 파견 클리커가 연결되는 최소 루프를 먼저 완성한다.

1. ~~메인 함선 뷰에 인터랙션 오브젝트 배치~~ ✓
2. ~~패널 플립 전환 시스템 구현~~ ✓
3. ~~직접 파견 클리커 화면 — 외계생물 등장, 클릭 데미지, 처치, 크레딧 드랍~~ ✓
4. ~~복귀/정산 구조 결정 후 격납고 패널 구현~~ ✓
5. ~~크레딧 상태 표시 (game_state.gd 기반)~~ ✓

### 2순위 — 기본 성장 루프

직접 파견으로 번 크레딧을 쓸 수 있는 최소 구조.

1. PC 터미널 패널 — 파츠 구매, 파일럿 고용 기본 UI
2. 공작실 패널 — 파츠 조합으로 머신 완성
3. 파견 관제 패널 — 임무 보드, 파일럿 배정, 자동 파견 시작

### 3순위 — 확장

1. 행성 해금 시스템 (크레딧 지불)
2. 플레이어 기체 업그레이드 (클릭 데미지, 클릭 범위)
3. 자동 재파견 및 전원 출격
