# 파견 루프 시스템

## 목적

이 문서는 `Dispatch Clicker`의 가장 작은 플레이 단위인 파견 루프를 분리해 정리한다.

## 현재 정의

파견 루프는 아래 흐름으로 동작한다.

1. 플레이어가 지역을 선택한다.
2. 플레이어 또는 자동 유닛이 파견된다.
3. 파견 중 임무가 진행된다.
4. 플레이어가 수동 복귀하거나, 일정 시간이 지나 자동 복귀한다.
5. 복귀 시 성과와 시설 상태를 기준으로 정산한다.
6. 수익을 다음 성장 요소에 재투자한다.

## 구현된 기능

MVP 및 이후 단계 구현 완료:

- 플레이어 직접 파견 + 클릭 기반 임무 진행 (행성 선택, 웨이브, 크레딧 정산)
- 수동/웨이브 완료 자동 복귀
- 자동 파견 유닛 루프 (3슬롯, 머신+파일럿 배정, 타이머 기반 임무/귀환)
- 파일럿 보너스(speed/credits) 파견 연동
- 저장/불러오기, 오프라인 진행 계산 (단일 사이클)
- 자동 재파견 (백엔드 구현, UI 비활성)

## 추후 확장

- 자동 재파견 다중 사이클 오프라인 계산
- 피로도 / 정비 대기
- 희귀 보상 / 특수 이벤트
- 전원 출격 버튼
- 자동 파견 수익 상세 내역
- CCTV 모니터링 창
