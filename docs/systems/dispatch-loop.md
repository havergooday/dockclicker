# 파견 루프 시스템

## 목적

이 문서는 `Dispatch Clicker`의 가장 작은 플레이 단위인 파견 루프를 분리해 정리한다.

## 현재 정의

파견 루프는 아래 흐름으로 동작한다.

1. 플레이어가 지역을 선택한다.
2. 플레이어 또는 자동 유닛이 파견된다.
3. 파견 중 임무가 진행된다.
4. 플레이어가 수동 복귀하거나, 일정 시간이 지나 자동 복귀한다.
5. 복귀 시 성과와 시설 상태를 기준으로 정산한다.
6. 수익을 다음 성장 요소에 재투자한다.

## MVP 범위

첫 MVP에서는 아래만 포함한다.

- 플레이어 직접 파견
- 클릭 기반 임무 진행
- 수동 또는 시간 기반 복귀
- 단순 정산
- 크레딧 반영

## 추후 확장

- 지역별 특성 반영
- 머신/파일럿 스탯 반영
- 자동 파견 유닛 루프
- 피로도 / 정비 대기
- 희귀 보상 / 특수 이벤트
