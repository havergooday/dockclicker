# 구현된 기능

## 개요

이 문서는 현재 저장소에 실제로 반영된 기능, 구조, 문서를 정리한다.  
여기서 말하는 `구현된 기능`은 플레이 가능한 게임 기능뿐 아니라, 프로젝트 운영에 필요한 문서/구조/검증도 포함한다.

- **카테고리 사이드바** — 클릭강화 / 파일럿 / 몸체 / 무기 / 다리 / 보유파츠 (6개)

## 공작실 패널 UI 리뉴얼 완료

### 파츠목록 탭
- 파일럿·몸체·무기·다리 전 파츠 그리드 표시 (티어별 컬러 보더 정사각형 박스)
- 박스 클릭 → 우측 상세 패널에 파츠 이름·효과·보유수량 표시
- 보유 수량 0이면 박스 반투명 처리

### 파츠조립 탭
- **3열 레이아웃** (1:2:2 비율) — 격납고 / LOADOUT / SYS SPEC
- **격납고 열**: 슬롯 버튼 배열 (52px 직사각형), 상태별 색상, 잠금/비어있음/대기/파견중 표시
- **LOADOUT 열**: 파츠 타입별 섹션 카드 + 직사각형 아이템 박스 배치, 클릭으로 선택
- **SYS SPEC 열**: RPG 장비 슬롯 (파일럿·몸체·무기·다리 개별 슬롯), 라이브 스펙 (임무시간·복귀시간·예상수익·CR/s), 상태 표시 (조립 버튼 or 안내 문구)
- 격납고에서 빈 슬롯 클릭 → 공작실 이동 + 해당 슬롯 자동 선택 (`workshop_preselect_slot`)

- 잠금 슬롯: 해금 비용 표시 + 해금 버튼

## 파견 관제 패널 UI 리뉴얼 완료 (B형 레이아웃)

### 상단 직접출격 스트립
- 행성 버튼 가로 나열 (선택 시 보라색 강조), 미해금 행성 잠금 표시 + 해금 비용
- 맨 우측 "직접 출격 ▶▶" 버튼

### 하단 슬롯 카드 3개 (1600px 균등 분할)
- 카드 상단 3px 보더: 상태별 색상 구분
  - 🔒 잠금(회색), 비어있음(청회색), 대기중(파랑), 파견중(주황), 귀환중(시안), 귀환완료(초록)
- **잠금 카드**: 해금 비용 + 해금 버튼
- **비어있음 카드**: 공작실 이동 버튼
- **대기중 카드**: 파일럿 선택 → 지역 선택 → 출격 (인라인 순차 표시)
  - 파일럿 선택 시 다른 슬롯 카드 자동 초기화
- **파견중/귀환중 카드**: 실시간 프로그레스바 + MM:SS 카운트다운 + 예상 수익
- **귀환완료 카드**: 획득 CR + 수령 버튼
- 격납고에서 offline 슬롯 선택 → 파견 관제 진입 시 해당 카드 자동 확장 (`dispatch_preselect_slot`)

## 자동 파견 타이머 시스템 구현 완료

- **임무 단계** (`on_mission`): 몸체 파츠 가치(초) 동안 진행
- **귀환 단계** (`returning`): 기본 30초 − 다리 파츠 감소값 (최소 5초)
- **복귀완료** (`returned`): 수동 수령 필요
  - 수령 시: 무기 CR/s × 임무 시간 크레딧 지급, 파일럿 인벤토리 반환, 슬롯 offline 복귀
- `DispatchManager._process()` — 매 프레임 타이머 체크, 상태 자동 전환
- `get_machine_preview()` — 머신 스펙 미리보기 (임무시간·복귀시간·예상수익·CR/s)

## 브릿지 — 파견 현황 표시 구현 완료

- 브릿지 하단에 파견 현황 상시 표시
- 임무중 슬롯: 슬롯 번호·목적지 + 임무 완료까지 카운트다운 (MM:SS)
- 귀환중 슬롯: 시안 + 복귀까지 카운트다운 (MM:SS)
- 파견 없음: 흐린 안내 문구
