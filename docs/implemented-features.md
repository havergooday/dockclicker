# 구현된 기능

## 개요

이 문서는 현재 저장소에 실제로 반영된 기능, 구조, 문서를 정리한다.  
여기서 말하는 `구현된 기능`은 플레이 가능한 게임 기능뿐 아니라, 프로젝트 운영에 필요한 문서/구조/검증도 포함한다.  
현재 저장소는 기능 추가를 멈춘 고정 상태이며, 이 문서는 그 기준선을 기록하는 용도다.

## 브릿지 — 로밍 파일럿 구현 완료

- 고용된 파일럿이 브릿지 화면을 자유롭게 돌아다니는 애니메이션
- **구현 방식**: Tween 체인 — 이동 → 깊이 업데이트 → 1.5~4초 대기 → 반복 (`_process()` 없음)
- **원근감**: Y 좌표 기반 scale(0.75~1.0) + z_index 변화로 공간감 연출
- **컬러 사각형 플레이스홀더** (56×96px) — 파일럿별 커스텀 색상, 픽셀 아트 스프라이트 교체 예정
- **클릭 말풍선** — 파일럿 클릭 시 이름·상태 말풍선 3.5초 표시 후 자동 소멸
- **파견 연동**: 파견 중 페이드 아웃 + 이동 일시 정지, 귀환 시 복귀
