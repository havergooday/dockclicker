## 탐색 가이드

- 게임 시스템 이해 → `기획서`
- 화면 구조/비주얼 → `디자인 문서`
- 무엇이 만들어졌는지 → `구현된 기능` + `프로젝트 정보`
- 다음 작업 → `현재 우선순위`
- 기술 선택 근거 → `기술 결정 기록`
- 커밋 규칙 → `커밋 워크플로우`
- 특정 시스템 상세 → `시스템 문서`
