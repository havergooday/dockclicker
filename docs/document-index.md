# 문서 인덱스

## 개요

`Dispatch Clicker` 프로젝트의 문서 허브다.  
현재 저장소는 기능 추가를 멈추고, 버그 수정과 레거시 삭제를 우선하는 고정 상태로 운영한다.  
AI 컨텍스트는 프로젝트 루트 `CLAUDE.md`에 있다.

- [현재 우선순위](current-priorities.md) - 지금 진행할 유지보수 순서

## 탐색 가이드

- 게임 시스템 이해 -> `기획서`
- 화면 구조/비주얼 -> `디자인 문서`
- 무엇이 만들어졌는지 -> `구현된 기능`
- 지금 할 일 -> `현재 우선순위`
- 기술 선택 근거 -> `기술 결정 기록`
- 특정 시스템 상세 -> `시스템 문서`
