# Dockclicker First Screen Design

**Goal**

브라우저 한쪽에 띄워두고 가끔 클릭할 수 있는 아주 작은 클리커의 첫 화면을 만든다. 지금 단계에서는 게임성보다 화면 밀도와 기본 상호작용 확인이 목적이다.

**Scope**

- 점수 숫자 표시
- 큰 클릭 버튼 1개
- 클릭 시 점수 `+1`
- `localStorage`를 이용한 점수 저장/복원
- 정적 HTML/CSS/JS 구조

**Out of Scope**

- 자동 수익
- 업그레이드
- 애니메이션 중심 연출
- 테마/컨셉 디자인 확정
- 백엔드 또는 계정 동기화

**Structure**

- `index.html`: 기본 레이아웃과 스크립트 연결
- `styles.css`: 좁은 세로 영역에서도 보이는 최소 UI 스타일
- `app-state.mjs`: 점수 계산과 저장/복원 로직
- `app.mjs`: DOM 렌더링과 클릭 이벤트 연결

**UI**

- 화면 중앙에 카드형 패널 1개
- 상단에 현재 점수
- 중앙에 큰 버튼
- 설명 문구는 최소화

**Behavior**

- 첫 진입 시 저장된 점수를 읽어 표시
- 저장값이 없거나 잘못된 값이면 `0`으로 시작
- 버튼 클릭 때마다 점수를 올리고 즉시 저장

**Testing**

- 상태 저장/복원 로직을 Node 내장 테스트로 검증
- 브라우저 전용 DOM 동작은 수동 확인
