[ 게임플레이 코어 ]

01. 단계별 기능 해금 프로그레션
    처음엔 클리커만 → CR 100 달성 시 상점, CR 500 시 격납고 등 자연스러운
    게이트가 없으면 빈 화면에 방치되는 느낌

02. 클릭 데미지 숫자 팝업
    클릭 위치에 "+12" 플로팅 텍스트 하나만으로 타격감이 완전히 달라짐

03. 클릭 범위 업그레이드
    PC 터미널에서 구매 가능한 "범위 클릭"은 단일→군집 플레이 방식 전환이라
    중반 게임체인저

04. 클릭 콤보 시스템
    연속 클릭 N회마다 배율 보너스 — 마우스를 빨리 누를 이유가 생김

05. 행성별 고유 생물 2-3종
    👾 단일 이모지로만 채워진 클리커 화면의 가장 큰 구멍,
    크기·속도·HP 다른 생물만 있어도 판이해짐

06. 파츠 드랍 (클리커에서)
    고난이도 행성 처치 시 확률적 파츠 획득 —
    직접 파견 동기를 크레딧 외로 확장

07. 클릭 임팩트 이펙트
    히트 시 작은 파티클 + 피격 흔들림,
    없으면 클릭이 반응 없는 버튼 누르는 느낌

--------------------------------------------------------------------------------

[ 자동 파견 / 운용 ]

08. 전원 출격 버튼
    베이가 늘어날수록 하나씩 클릭하는 게 노동이 됨, 핵심 QoL

09. 오프라인 다중 사이클 계산
    현재 1사이클만 계산해서 오래 떠나면 손해 — 방치형 게임의 기본

10. 자동 재파견 토글
    베이별 귀환 즉시 재출격 설정 —
    완전 방치 운영 가능해지는 전환점

11. 자동 파견 수익 예측
    "현재 배치로 시간당 예상 XXX CR" 한 줄이
    유저에게 최적화 방향 제시

12. 파견 보고서
    귀환 시 처치 수·소요 시간·특이사항 짧은 텍스트 —
    파견이 의미 있는 사건처럼 느껴짐

13. 파견 수익 내역 상세
    수령 시 "처치 42마리 × 10CR + 보너스" 구성 내역,
    지금은 숫자만 보임

[ 파일럿 ]

14. 파일럿 레벨업
    파견 횟수 누적 → 경험치 → 레벨업 → 보너스 강화,
    고용 후 애착이 생기는 구조

15. 파일럿 전문 행성 보너스
    특정 파일럿이 특정 행성에서 보너스 —
    배치 최적화가 메타게임이 됨

16. 파일럿 임무 일지
    귀환 후 짧은 랜덤 텍스트 로그 —
    파일럿이 캐릭터처럼 느껴지는 가장 저비용 방법

17. 파일럿 피로도
    연속 파견 시 효율 감소, 대기 중 회복 —
    로테이션 전략 레이어 추가

--------------------------------------------------------------------------------

[ 성장 / 진행 ]

18. 행성 해금 연출
    지금 해금해도 아무 반응 없음,
    슬라이드+반짝 효과 하나로 "달성감" 완성

19. 행성 난이도 미리보기
    잠긴 행성에도 적 HP·보상 표시 —
    해금 욕구를 구체적 목표로 전환

20. 파츠 분해 → 크레딧 환수
    쌓인 저티어 파츠 처리 방법,
    없으면 인벤토리가 쓰레기통

21. 머신 이름 부여
    조립 시 이름 입력 한 줄 —
    "1번 베이 머신"이 "불사조-1"이 되는 순간 몰입도 전환

22. 머신 프리셋
    자주 쓰는 파츠 조합 저장/불러오기,
    베이 많아지면 반복 조립이 지루해짐

23. 시설 업그레이드 트리
    격납고 수용량, 파견 속도 등 실질 수치 투자 트리 —
    크레딧 써야 할 이유 추가

--------------------------------------------------------------------------------

[ 비주얼 / UI ]

24. 메크 스프라이트 조합
    파츠 조합별 시각적 머신 실루엣,
    조립 행위가 "만드는 것"처럼 느껴지는 핵심 연출

25. SD 파일럿 스프라이트
    브릿지 원형 플레이스홀더 → 픽셀 아트 캐릭터,
    브릿지가 살아있는 공간으로 전환

26. 크레딧 HUD 롤업 애니메이션
    수령 시 숫자가 올라가는 효과 하나가 "보상감"의 거의 전부

27. 글로벌 파견 현황 배지
    브릿지에 "파견 중 4 / 귀환 대기 2" 상시 표시 —
    상태 파악을 위해 격납고 갈 필요 없어짐

[ 시스템 / 플랫폼 ]

28. BGM / SFX
    클릭음·수령음·파견음 세 개만으로도 게임 느낌 완전히 다름

29. 귀환 완료 OS 알림
    데스크톱 컴패니언 특성 최대 활용 —
    창 안 보고 있어도 귀환 알림

30. 파츠 iid 기반 조립 선택
    같은 티어 파츠 중 특정 인스턴스를 구별 선택,
    옵션 시스템 도입 전 필수 기반
