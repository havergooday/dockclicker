# 아직 없는 기능

## 개요

아직 구현되지 않았지만 현재 기획 기준으로 만들어야 할 기능을 정리한다.  
설계가 확정된 항목과 아직 논의 중인 항목을 구분해서 표시한다.

## 핵심 UI 시스템 (설계 확정)

- SD 캐릭터 idle 애니메이션 (메인 함선 뷰)
- 파견 중 유닛 상태 표시 (메인 함선 뷰)

- 플레이어 기체 업그레이드 — 클릭 데미지, 클릭 범위, 무기 종류

## 행성/지역 시스템 (설계 확정)

- 기본 행성 1개에서 시작
- 크레딧 지불로 새 행성 해금
- 행성별 생물 종류 및 보상 품질 차별화
- 고난이도 행성 머신 파츠 드랍

## 패널 UI (설계 확정)

- **격납고 패널** — 귀환 유닛 수령, 크레딧 획득 인터랙션, 머신 상태 확인
- **공작실 패널** — 보유 파츠로 머신 조립 인터페이스
- **PC 터미널 패널** — 파츠 구매 + 파일럿 고용 상점 UI
- **파견 관제 패널** — 임무 보드(카드 목록), 파일럿 담당 행성 배정, 자동 재파견 설정, 전원 출격, 직접 출격

## 자동 파견 시스템

- 파일럿 + 머신 = 자동 파견 유닛 구성
- 담당 행성 지정 및 자동 재파견
- 휴식/정비 대기 시간
- 전원 출격 버튼

## 머신 및 파일럿

- 머신 파츠 구매 (PC 터미널)
- 파일럿 고용 (PC 터미널)
- 파츠 조합으로 머신 완성 (공작실)
- 파일럿-머신 배정

## 성장 시스템

- 시설 업그레이드
- 기지 확장
- 지역 해금 (크레딧 구매)
- 파견 효율 상승

## 자동 파견 모니터링 (MVP 이후)

- CCTV 소형 창 — 저해상도/노이즈 필터 느낌의 작업 현황 확인
- 구체적 표시 방식(상태바, 픽셀 뷰, 텍스트 피드)은 추후 결정

## 데이터/저장

- 파견 지역 데이터
- 머신/파일럿 데이터
- 업그레이드 데이터
- 저장/불러오기
- 오프라인 진행 계산

## 시스템 품질

- 실제 Godot 에디터 기준 실행 확인
- 창 크기/비율 대응 (가로형/세로형)
- 저자원 동작 최적화
- 스팀 배포용 설정
