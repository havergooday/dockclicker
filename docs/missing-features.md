# 아직 없는 기능

## 개요

아직 구현되지 않았지만 현재 기획 기준으로 만들어야 할 기능을 정리한다.

## 핵심 UI 시스템

- **UI/UX 통합 재설계** — 파노라마 함선 캔버스 진행 중 (`docs/plans/2026-05-27-ux-redesign.md`)
  - 완료: `ShipCanvas` 메인 루트, 관제실 `항성지도` 팝업, `HangarZone` 베이 그리드 + 오버레이 팝업, `ShopPopup` 파일럿 공고판, `PartsShopPopup` 파츠 구매 팝업, `FacilityManagementPopup` 시설관리 팝업, 브릿지 파일럿 로밍, 격납고 조립 팝업
  - 남은 작업: 관제실/격납고 UI 미세 조정, 침대/라운지 배치 폴리싱
- SD 캐릭터 idle 애니메이션 — 현재 컬러 사각형 플레이스홀더, 픽셀 아트 스프라이트 미구현
- ~~상태 표시줄 위치 옵션 (상/하/좌/우, 설정에서 변경)~~ ✓ 구현 완료 — `hud_position` + 옵션 팝업 토글, 좌/우 세로 레이아웃

## 직접 파견 클리커 (추가 개선)

- 행성별 고유 외계생물 실제 픽셀 아트 에셋 교체 (MVP는 region_type별 코드 기반 생물 비주얼로 대체 완료)
- 플레이어 기체 무기 종류 변경 (클릭 범위·콤보·자동공격은 구현 완료)

## 파츠 옵션 시스템

- ~~티어 동적 결정~~ ✓ 구현 완료 — 옵션 0개=T1 / 1개=T2 / 2개=T3, 드랍 시 행성 등급에 따라 옵션 수 결정
- 설계 원칙은 `docs/dispatch-clicker-design.md` 파츠 옵션 시스템 섹션 참조
- 참고: 드랍 파츠 옵션 부착 / 효과 적용 / 파츠 분해 환급은 구현 완료
- 참고: 공작실·베이 팝업 모두 iid 기반 인스턴스 단위 선택 구현 완료 (동일 티어 옵션 구분)

## 완료된 균형 조정 (2026-05-31)

- 시작 크레딧: 100 → 500
- 공작실 해금 비용: 1,000 → 800
- 조립 비용: tier×50 → tier×40
- 침대/시설 회복 틱: 5초 → 4초
- EXP 승급 임계값: [100, 200] → [80, 160]
- 진행 힌트 레이블 추가 (브릿지 화면, 미션 단계별 자동 표시)
