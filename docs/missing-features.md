# 아직 없는 기능

## 개요

아직 구현되지 않았지만 현재 기획 기준으로 만들어야 할 기능을 정리한다.

## 핵심 UI 시스템

- SD 캐릭터 idle 애니메이션 (메인 함선 뷰)

## 직접 파견 클리커 (추가 개선)

- 행성별 고유 외계생물 그래픽/종류 (현재 👾 단일)
- 플레이어 기체 업그레이드 추가 — 클릭 범위, 무기 종류
- 고난이도 행성 머신 파츠 드랍

## 자동 파견 — 미구현 기능

- 자동 재파견 (귀환 후 즉시 재출격 설정)
- 전원 출격 버튼
- 자동 파견 수익 상세 내역 표시

## 자동 파견 모니터링 (MVP 이후)

- CCTV 소형 창 — 저해상도/노이즈 필터 느낌의 작업 현황 확인
- 구체적 표시 방식(상태바, 픽셀 뷰, 텍스트 피드)은 추후 결정

## 성장 시스템

- 시설 업그레이드
- 기지 확장
- 파견 효율 상승

## 데이터/저장 (전체 미구현)

- 저장/불러오기 (현재 런타임 메모리만, 종료 시 초기화)
- 오프라인 진행 계산

## 시스템 품질

- 창 크기/비율 대응 (가로형/세로형)
- 저자원 동작 최적화
- 스팀 배포용 설정
