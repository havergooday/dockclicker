# 아직 없는 기능

## 개요

현재 기준으로 남아 있는 미구현 또는 보류 항목만 적는다.  
이 저장소는 기능 추가를 멈춘 상태이므로, 아래 항목은 활성 로드맵이 아니라 현재 남아 있는 차이점 목록이다.

## 시각 에셋

- 실제 픽셀 아트 에셋 교체
- 행성별 고유 생물 실제 아트 교체
- 브릿지 파일럿 idle/픽셀 스프라이트
- 파일럿 커스텀 복장 그래픽
- 플레이어 기체 추가 업그레이드용 비주얼 에셋

- 관계 / 기억파편 개인 이벤트
