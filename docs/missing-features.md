# 아직 없는 기능

## 개요

이 문서는 아직 구현되지 않았지만, 현재 기획과 구조를 기준으로 앞으로 만들어야 할 기능을 정리한다.

## 코어 게임플레이 미구현

- 직접 파견 시작
- 파견 중 클릭 기반 임무 수행
- 수동 복귀
- 일정 시간 후 자동 복귀
- 복귀 후 정산
- 수익 획득 루프

## 자동화 시스템 미구현

- 머신 파츠 구매
- 머신 조립
- 파일럿 고용
- 파일럿 배정
- 자동 파견 유닛 생성
- 자동 파견 슬롯 운용
- 휴식/정비 대기 시간

## 성장 시스템 미구현

- 시설 업그레이드
- 기지 확장
- 플레이어 본체 강화
- 지역 해금
- 파견 효율 상승 구조

## UI/연출 미구현

- 실제 중앙 운영 패널 UI
- 격납고 슬롯 UI
- 브리지/라운지 상태 UI
- SD 캐릭터 표시
- SD 캐릭터 대기/이동/휴식 연출
- 가로형/세로형 실전 전환 로직
- 설정/로그/알림/사운드 유틸 UI

## 데이터/저장 미구현

- 파견 지역 데이터
- 머신 데이터
- 파일럿 데이터
- 업그레이드 데이터
- 저장/불러오기
- 오프라인 진행 계산

## 시스템 품질 관련 미구현

- 실제 Godot 에디터 기준 실행 확인
- 저자원 동작 최적화 설정
- 창 크기/비율 대응
- 스팀 배포용 설정
- 리소스 파이프라인 정리

## 우선순위가 높은 다음 단계

1. 메인 씬을 실제 UI 형태로 구체화
2. 직접 파견 MVP 루프 구현
3. 정산 및 수익 반영 구현
4. 저장 구조 설계
5. SD 캐릭터/브리지 상태 연출 최소 버전 추가
