# 아직 없는 기능

## 개요

아직 구현되지 않았지만 현재 기획 기준으로 만들어야 할 기능을 정리한다.  
설계가 확정된 항목과 아직 논의 중인 항목을 구분해서 표시한다.

## 핵심 UI 시스템

- SD 캐릭터 idle 애니메이션 (메인 함선 뷰)
- 파견 중 유닛 상태 표시 (메인 함선 뷰)

## 직접 파견 클리커 (추가 개선)

- 행성별 고유 외계생물 그래픽/종류 (현재 👾 단일)
- 플레이어 기체 업그레이드 추가 — 클릭 범위, 무기 종류
- 고난이도 행성 머신 파츠 드랍

## 행성/지역 시스템 (추가 개선)

- ~~기본 행성 1개에서 시작~~ ✓
- ~~크레딧 지불로 새 행성 해금~~ ✓
- ~~행성별 생물 수/보상 차별화~~ ✓ (수/보상 기반, 그래픽은 미구현)
- 고난이도 행성 머신 파츠 드랍

## 패널 UI

- **격납고 패널** — 자동 파견 유닛 슬롯 (직접 파견 슬롯은 구현됨)
- **공작실 패널** — 보유 파츠로 머신 조립 인터페이스 (전체 미구현)
- **PC 터미널 패널** — ~~클릭 데미지 강화~~ ✓ / 파츠 구매 + 파일럿 고용 UI 미구현
- **파견 관제 패널** — ~~직접 출격 버튼~~ ✓ ~~행성 선택~~ ✓ / 임무 보드, 파일럿 배정, 자동 재파견 미구현

## 자동 파견 시스템 (전체 미구현)

- 파일럿 + 머신 = 자동 파견 유닛 구성
- 담당 행성 지정 및 자동 재파견
- 휴식/정비 대기 시간
- 전원 출격 버튼
- 자동 파견 유닛 귀환 슬롯 (격납고)

## 머신 및 파일럿 (전체 미구현)

- 머신 파츠 구매 (PC 터미널)
- 파일럿 고용 (PC 터미널)
- 파츠 조합으로 머신 완성 (공작실)
- 파일럿-머신 배정

## 성장 시스템

- 시설 업그레이드
- 기지 확장
- 파견 효율 상승

## 자동 파견 모니터링 (MVP 이후)

- CCTV 소형 창 — 저해상도/노이즈 필터 느낌의 작업 현황 확인
- 구체적 표시 방식(상태바, 픽셀 뷰, 텍스트 피드)은 추후 결정

## 데이터/저장 (전체 미구현)

- 저장/불러오기 (현재 런타임 메모리만, 종료 시 초기화)
- 머신/파일럿 데이터
- 오프라인 진행 계산

## 시스템 품질

- 실제 Godot 에디터 기준 실행 확인 및 버그 수정
- 창 크기/비율 대응 (가로형/세로형)
- 저자원 동작 최적화
- 스팀 배포용 설정
